Remember to name post files in the following format:

`YEAR-MONTH-DAY-title.markdown`

Where `YEAR` is a four-digit number, `MONTH` and `DAY` are both two-digit numbers.

Code snippet examples:

{% highlight ruby %}
def print_hi(name)
  puts "Hi, #{name}"
end
