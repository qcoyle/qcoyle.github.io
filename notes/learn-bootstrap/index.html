Terms

Bootstrap

  • A CSS framework with prewritten CSS rules to help build web pages faster and nicer
  • Makes creating websites quicker
  • One of the most popular front end frameworks
  • Made by Twitter people

Bootstrap features

  • Keeps item orientation pretty even when changing page size

Bootstrap pros and cons

  • Pros
  • Well-tested across browsers
  • Lightweight
  • Includes JS plugs using jQuery
  • Lots of templates and wordpress plugins
  • If you don’t have a lot of time/money for designers/developers, use frameworks like Bootstrap. As you scale, might need development assistance
  • Cons
  • Restricts your creativity
  • Lots of code time if you deviate from customary Bootstrap website design
  • Can lead to lots of HTML output

Linking to bootstrap

  • In addition to main.css, need to link to a url that hosts bootstrap
  • I.e.

<link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/3.3.6/css/bootstrap.min.css"/>

Grid

  • Bootstrap uses grids to organize html using preconfigured columns
  • Bootstrap’s grid has 12 columns of equal sizes. Elements get arranged to span different numbers of columns
  • Bootstrap’s grid is responsive

Incorporating bootstrap

  • Include the following to HTML:

<head>

   <!-- Required meta tags -->

   <meta charset="utf-8">

   <meta name="viewport" content="width=device-width, initial-scale=1, shrink-to-fit=no">

   <!-- Bootstrap CSS for styling and layout-->

   <link rel="stylesheet" href="https://stackpath.bootstrapcdn.com/bootstrap/4.1.3/css/bootstrap.min.css"

       integrity="sha384-MCw98/SFnGE8fJT3GXwEOngsV7Zt27NXFoaoApmYm81iuXoPkFOJwJ8ERdknLPMO" crossorigin="anonymous">

</head>

  • Note: there are also some JavaScript libraries on Bootstrap’s getting started page if you want some interactivity

container

  • The parent element of Bootstrap’s grid (which is based off of the Flexbox)
  • Rows are the immediate children, then columns are nested within rows
  • Actual content goes in the column elements
  • Equal margin on left and right
  • To incorporate, set the HTML class to “container”

container-fluid

  • Assign this class in HTML if you want the container to take up the whole screen width

row

  • A container to use responsive columns
  • We can add as many rows as we want
  • Assign this class of row to an HTML element

col

  • Column: The immediate children of rows in Bootstrap
  • They store content
  • Takes up the entire width of its parent row by default. Adding multiple columns make each column the same width by default
  • Max 12 columns in a row

        

col-*-*

  • A responsive grid to span a column
  • First *: the breakpoint. See The five breakpoints of Bootstrap
  • Note: There is no xs. That’s the default if it’s omitted:

<div class="col-8">

   <p>This is the width of 8 columns.</p>

 </div>

  • Second *: Values can be 1-12 for columns to span
  • For instance, the below effect takes place on all screens small and larger

<div class="col-sm-3">

  • To combine declarations:

<div class="col-12 col-md-8 col-xl-6">

</div>

Example basic syntax of a row and column

<body>

   <div class="container">

       <div class="row">

               <h3>Testing!</h3>

           </div>

       </div>

   </div>

<body>

col-auto

  • Use this in the HTML to use the content (i.e. the text) to set the column’s width

Text utility syntax

text-*

  • Change text color. From Bootstrap docs:
  • <p class="text-primary">My text</p>

bg-*

  • Change background color. From Bootstrap docs:
  • <div class="bg-success">This element is green</div>
  • Note: this can apply to more than just text

font-weight-*

  • Make text bold, light, or italic
  • From docs:
  • <p class="font-weight-bold">Bold text.</p>
  • <p class="font-weight-bolder">Bolder weight text (relative to the parent element).</p>
  • <p class="font-weight-normal">Normal weight text.</p>
  • <p class="font-weight-light">Light weight text.</p>
  • <p class="font-weight-lighter">Lighter weight text (relative to the parent element).</p>
  • <p class="font-italic">Italic text.</p>

text-uppercase

  • Make text all uppercase
  • Example of bold and uppercase

<p class="font-weight-bold text-uppercase">

   This rendered text is both bold and uppercased.

</p>

text-*

  • Aligns text, i.e. right

Interactivity syntax

fixed-*

  • Fix element on certain side of screen, i.e. top

nav

  • Give the user a collection of links to navigate

<div class="container">

   <!-- Row containing nav component -->

   <div class="row">

       <div class="col bg-secondary">

           <!-- Manipulate the elements below: -->

           <nav class="nav justify-content-center">

               <a class="nav-link" href="#">About Us</a>

               <a class="nav-link" href="#">Contact Us</a>

               <a class="nav-link" href="#">Subscribe</a>

           </nav>

   </div>

   <!-- End of row containing nav component -->

</div>

btn btn-*

  • Style page elements as buttons
  • Example:

<div class="row">

       <div class="col bg-secondary">

           <button type="button" class="btn btn-danger">Danger</button>

           <button class="btn btn-success"> Success</button>

           <button type="button" class="btn btn-link">Link</button>

       </div>

   </div>

data-toggle="collapse"

  • Adding this tag to a button allows the div element below it to get toggled
  • Link data-target and data-toggle to corresponding class and id name in the element that gets toggled
  • aria-expanded and aria-controls control for screen readers/accessibility
  • Note that the associated Javascript is also required

card

  • A container for smaller custom content. They’re often grouped together to display a bunch of similar content, but not all at once
  • See example below
  • Example notes:
  • Card takes up the entire width of parent element by default
  • <img>

  • <div>
  • card-body class adds a section with default padding
  • The card-title, card-text, and card-link classes all add Bootstrap card styling to these elements

carousel

  • Showcase a group of images but let users see them in a slideshow format
  • See other examples in docs
  • Notes on example:
  • Parent <div>
  • id is used to add controls for clicking between slides
  • carousel and slide give style/formatting for carousel
  • data-ride makes the slides interactive
  • Nested <div> with class=”carousel-item”
  • active means the image that’s shown. Only one image should have the active class. If none have it, none will be shown
  • Nested <img> elements in <div> with class=”carousel-item”
  • d-block sets display to be block
  • w-100 says to take up 100% of the width
